# SmartBlog

A static blog landing page built with HTML and Tailwind CSS.

## Getting Started

1. Open `index.html` in your browser.
2. No build step is required because Tailwind CSS is loaded via CDN.

## What is included

- Hero section
- Feature cards
- Latest articles previews
- Newsletter subscription CTA
- Responsive layout
- User dashboard mockup
- Admin dashboard mockup

## Customize

Edit `index.html` to update text, add blog cards, or change colors.
